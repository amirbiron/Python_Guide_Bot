# ⚡ התחלה מהירה - בוט לימוד פייתון (20 שיעורים!)

## 🚀 תוך 5 דקות תהיה לך בוט עובד עם 20 שיעורים!

### צעד 1: קבל טוכן מ-BotFather (2 דקות)
1. פתח את אפליקציית טלגרם
2. חפש `@BotFather` ופתח שיחה
3. שלח: `/newbot`
4. תן שם לבוט (למשל: "Python Learning Bot")
5. תן username (למשל: "my_python_learn_bot")
6. **העתק את הטוכן** שמקבלים

### צעד 2: התקן (1 דקה)

**Windows:**
```bash
setup.bat
```

**Mac/Linux:**
```bash
chmod +x setup.sh
./setup.sh
```

או ידנית:
```bash
pip install -r requirements.txt
```

### צעד 3: הכנס את הטוכן (30 שניות)
פתח את `bot.py` ובשורה 8 שים את הטוכן:
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

### צעד 4: הרץ! (30 שניות)
```bash
python bot.py
```

תראה:
```
🤖 הבוט מתחיל...
📚 טוען 20 שיעורים...
✅ הבוט פועל! לחץ Ctrl+C לעצירה.
🎓 הקורס כולל 20 שיעורים מיסודות ועד מתקדם!
```

### צעד 5: נסה את הבוט! (30 שניות)
1. פתח את הבוט בטלגרם
2. שלח `/start`
3. לחץ "השיעור הנוכחי שלי"
4. תהנה מ-20 שיעורים מלאים! 🎉

## ❓ בעיות נפוצות

**"ModuleNotFoundError: No module named 'telegram'"**
→ הרץ: `pip install -r requirements.txt`

**"Unauthorized"**
→ הטוכן לא נכון, בדוק שהעתקת נכון

**"Conflict: terminated by other getUpdates"**
→ הבוט רץ פעמיים. עצור אחד (Ctrl+C)

**הבוט לא עונה**
→ בדוק שהקוד רץ (תראה "✅ הבוט פועל")
